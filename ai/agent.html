AI Agent ​

助手 ​

  • anthropics/claude-code，终端中的助手
  • esengine/DeepSeek-Reasonix，DeepSeek 原生的终端 AI 编程代理，高缓存命中，带思考。
  • pi/packages/coding-agent，不强制任何特定工作流，而是提供一个极简的终端基座，允许开发者通过扩展（Extensions）、技能（Skills）、提示模板（Prompt Templates） 和主题（Themes） 完全自定义行为。
  • lsdefine/GenericAgent，极简、可自我进化的自主 Agent 框架。不预设技能，靠进化获得能力
  • msitarzewski/agency-agents，各种用途的 Agent，相当于 system prompt
  • NousResearch/hermes-agent，能自我改进、从历史对话学习经验的 cli Agent
  • alibaba/page-agent，活在页面中的 Agent，页面中引入并配置模型、鉴权信息即可使用

助手的工具 ​

  • 轻舟 Agent Skill
  • 轻舟 MCP
  • OpenCLI，OpenCLI 作为一个 CLI 工具，通过 OpenCLI 浏览器扩展，来操控（登录态）浏览器（本地无需 AI，使用网站适配器已有的操控方式）。
