Python 字符串 ​

一、字符串基础 ​

操作代码示例
创建字符串s = "hello"s = 'hello' / s = ""
多行字符串s = """..."""三引号保留换行
长度len(s)len("Python") → 6
索引s[i]"Python"[0] → 'P', [-1] → 'n'
切片s[start:end:step]"Python"[1:4] → "yth"
反转s[::-1]"Python"[::-1] → "nohtyP"

二、常见方法 ​

大小写转换 ​

python
"hello".upper()        # "HELLO"
"HELLO".lower()        # "hello"
"hello world".title()  # "Hello World"
"hello".capitalize()   # "Hello"
"hElLo".swapcase()     # "HeLlO"

查找与判断 ​

python
s = "hello world"
s.find("world")        # 6（找不到返回 -1）
s.index("world")       # 6（找不到抛异常）
s.count("l")           # 3
s.startswith("he")     # True
s.endswith("ld")       # True
"abc123".isalpha()     # False（字母数字判断）

去除空白 ​

python
"  hi  ".strip()       # "hi"
"  hi  ".lstrip()      # "hi  "
"  hi  ".rstrip()      # "  hi"
"xxhixx".strip("x")    # "hi"

拆分与合并 ​

python
"a,b,c".split(",")     # ['a', 'b', 'c']
"a b c".split()        # ['a', 'b', 'c']（默认空白）
"-".join(["a","b","c"]) # "a-b-c"

替换 ​

python
"hello hello".replace("hello", "hi")      # "hi hi"
"hello hello".replace("hello", "hi", 1)   # "hi hello"

三、转义字符 ​

转义含义
\n换行
\t制表符
\\反斜杠
\'单引号
\"双引号
\r回车
\u4E2DUnicode（中文"中"）
\x48十六进制（'H'）
python
print("Line1\nLine2")          # 两行输出
print("C:\\Users\\Name")       # C:\Users\Name
print("\u2665")                # ♥

四、字符串格式化 ​

f-string（Python 3.6+，推荐） ​

python
name, age = "Alice", 25
f"My name is {name}, age {age}"           # 变量
f"{2 + 3}"                                 # 表达式
f"{age:3d}"                                # 宽度3，右对齐
f"{age:03d}"                               # 补零：025
f"{3.14159:.2f}"                           # 3.14
f"{0.25:%}"                                # 25.000000%

.format() 方法 ​

python
"{} {}".format("Hello", "World")           # Hello World
"{1} {0}".format("World", "Hello")         # Hello World
"{name} is {age}".format(name="Bob", age=20)

% 格式化（旧式） ​

python
"%s is %d" % ("Bob", 25)                   # Bob is 25
"%.2f" % 3.14159                           # 3.14

五、原始字符串与 f-string 配合 ​

原始字符串 r"" ​

python
r"C:\new\file.txt"        # 反斜杠不转义
r"\n\t\\"                 # 字面 \n\t\\

原始 f-string rf"" 或 fr"" ​

python
name = "data"
rf"C:\temp\{name}.txt"                    # C:\temp\data.txt
rf"\b{keyword}\b"                         # 正则表达式字面边界
rf"{{Hello {name}}}"                      # 字面花括号：{Hello data}

限制 ​

python
rf"C:\temp\"          ❌  # 不能以反斜杠结尾
rf"C:\temp\\"         ✅  # 正确写法

六、字符串与列表转换 ​

python
# 字符串 → 列表
list("abc")            # ['a', 'b', 'c']
"a,b,c".split(",")     # ['a', 'b', 'c']

# 列表 → 字符串
"".join(['a','b','c']) # "abc"
",".join(['a','b','c']) # "a,b,c"

七、字符串不可变性 ​

python
s = "hello"
# s[0] = "H"          ❌ 报错：不支持修改
s = "H" + s[1:]       ✅ 创建新字符串："Hello"

八、常用判断方法 ​

方法说明示例
s.isalpha()全是字母"abc".isalpha() → True
s.isdigit()全是数字"123".isdigit() → True
s.isalnum()字母或数字"abc123".isalnum() → True
s.isspace()全是空白" \t".isspace() → True
s.islower()全是小写"abc".islower() → True
s.isupper()全是大写"ABC".isupper() → True

九、实用技巧 ​

多行字符串续行 ​

python
# 方法1：三引号
s = """第一行
第二行"""

# 方法2：反斜杠续行（不产生换行）
s = "这是一个很长很长的\
字符串"

检查子串 ​

python
if "world" in "hello world":    # True
if "abc" not in "hello":        # True

字符串乘法 ​

python
"*" * 10          # "**********"
"-=" * 3          # "-=-=-="

十、最佳实践建议 ​

场景推荐写法
文件路径（Windows）rf"C:\Users\{name}\\" 或 pathlib
正则表达式rf"\d+\.\d+"
包含引号交替使用 ' " ' 或用三引号
多行字符串三引号 """..."""
简单拼接f-string > + > .format()
性能要求高"".join(list) 比循环 + 快

十一、常见错误速查 ​

错误代码原因正确写法
s[0] = 'a'字符串不可变s = 'a' + s[1:]
s.find('x') 返回 -1找不到子串检查 if 'x' in s:
rf"path\"不能以 \ 结尾rf"path\\" 或 f"path\\"
s.index('x') 异常找不到子串先用 in 判断或用 find
