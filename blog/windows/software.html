Windows 软件 ​

系统镜像 ​

  • Atlas-OS/Atlas，Windows 修改版，增强隐私，优化性能，移除广告
  • ReviOS，搭配 Playbook 系统修改器，实现一系列自动优化、精简

TIP

使用方法

关闭 Windows Defender，安装 AME Wizard，添加 Atlas/ReviOS 的 PlayBook 文件，并运行

提醒 ​

FadeTop：提醒休息，很老的软件但是支持最新系统，晃动鼠标可以快速退出休息

生产力 ​

EcoPasteHub/EcoPaste: 🎉跨平台的剪贴板管理工具：剪切板

ShareX：长截图不错。默认设置截图会上传需关闭⚠️

PixPin：截图贴图（类似 Snipaste），支持长截图、文字识别、GIF 录制等。

ScreenToGif：录制 GIF，可导出视频

ScreenCapture：截图、贴图、标柱，推荐

uTools：工具集（类似 Quicker），支持搜索、翻译、OCR、插件市场等

Diffinity：文件夹各文件差异对比

Beyond Compare：文件对比工具

Bruno：api 请求调试（类似 postman），集合中的文件夹可加公共 header 字段和公共变量

开发 ​

Visual Studio Code：编辑器，支持各种语言，插件丰富，启动快

Starship：终端提示，配合 Nerd Font 使用，对终端启动速度影响小

文档 ​

Sumatra：PDF 阅读器，速度快，标注能力等于没有

Koodo Reader: PDF 阅读器，速度中等，标注能力中下

ONLYOFFICE Desktop Editors：office 替代品

Office Tool Plus：一键部署 Microsoft Office

Microsoft-Activation-Scripts：office 激活（ohook 原理），windows 也可激活

多媒体 ​

ImageGlass：图片查看，支持查看当前文件夹、一键设为壁纸和锁屏（不需要自己缩放壁纸）

mpv，开源跨平台媒体播放器，快捷键，必看，Applications using mpv

VLC：开源多媒体播放器

YesPlayMusic：网易云音乐第三方客户端，高颜值

下载 ​

Gopeed：磁力、种子下载，支持自动更新 tracker list

其他 ​

NanaZip，7z 现代化版本

Monitorian：开源屏幕亮度调节工具，支持多屏幕同时调节
