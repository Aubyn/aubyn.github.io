sing-box ​

是什么 ​

sing-box 也是一款正向代理工具。相比 mihomo，支持更丰富的配置。

为什么 ​

缺点：

  • 不支持负载均衡，支持 urltest 伪负载均衡
  • 配置项过于复杂，且经常变动，难以书写

优点：

  • dns/route 等支持更加复杂的规则
  • 性能好，路由比较快

如何使用 ​

  1. 准备 sing-box 工作目录，下方所有文件都放入此目录
  2. 下载 sing-box 二进制 Releases，选择对应的版本，解压
  3. 准备 ip 数据库 cnip.srs MetaCubeX/meta-rules-dat
  4. 准备配置文件 config.json，从代理提供商订阅链接或其他方式获取
  5. 管理员模式启动终端（tun 模式必须），进入工作目录，运行 .\sing-box.exe run -D . 启动
